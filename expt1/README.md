## 1. 数据集介绍
### 1.1 数据集来源
Adult数据集是一个关于人口普查的数据集，包含了48842个样本，每个样本包含14个特征，其中包括年龄、工作类型、教育程度、婚姻状况、职业、种族、性别、资产所得、资产损失、周工作时长、原籍、收入等信息。该数据集是一个二分类数据集，目标是预测一个人的收入是否超过5万美元/年。
